# Notebooks

In diesem Ordner werden die Parameterstudien und Referenzfälle für
`groundfield` abgelegt. Jedes Notebook steht für sich: Bodenmodell,
Geometrie und Frequenzraster werden oben definiert, am Ende liegt ein
reduziertes `rho-f`-Modell vor, das in `groundinsight` importiert
werden kann.

Geplante Notebooks entlang Arbeitspaket 1:

- `ap1_ring_homogen.ipynb` — Ringerder, homogenes Erdreich,
  analytische Referenz (Sunde).
- `ap1_ring_zweischicht.ipynb` — Parameterscan über $\rho_1$,
  $\rho_2$, $h_1$.
- `ap1_tn_ortsnetz.ipynb` — Ortsnetzstation + Hausanschlüsse +
  Kabelverteiler, PEN und MS-Kabelschirm.
- `ap1_messung_vs_fehlerfall.ipynb` — Gegenüberstellung Mess-
  konfiguration und Fehlerfall.

Die Notebooks werden erst beim Aufbau der jeweiligen Module produktiv;
bis dahin dient diese README als Platzhalter.
